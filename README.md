# Convertor

Instructions pour convertir des fichiers Word en fichiers JSON compatibles avec l'application :

- Téléchargez le fichier "WordJsonConverter.xlsm" sur votre ordinateur.
- Ouvrez le fichier "WordJsonConverter.xlsm" avec Excel.
- Dans la cellule B2, copiez le chemin d'accès au dossier contenant les fichiers Word à convertir.
- Remarque : Ce dossier doit uniquement contenir les fichiers Word que vous souhaitez convertir.
- Dans la cellule B3, copiez le chemin d'accès au dossier où vous souhaitez enregistrer les fichiers JSON.
- Remarque : Ce dossier de destination doit être vide.
- Cliquez sur le bouton "Générer les fichiers JSON" pour lancer le processus de conversion.
- Une fois les fichiers générés, copiez les dans le dossier src/Shared/Ressources de l'application, similairement aux fichiers actuels.
